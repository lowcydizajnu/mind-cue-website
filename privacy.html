3. How we use artificial intelligence

We use Claude, an AI model by Anthropic, to classify calendar events and personalise technique recommendations. AI features are opt-in only and gated by your explicit consent.

We send only the minimum data necessary — event titles, participant names, participant counts, and anonymised preference summaries. We never send your name, email, full journal history, or health data to Anthropic.

Anthropic's API does not use data submitted via the API to train its models. See anthropic.com/privacy.

7. Your rights under GDPR

If you are in the EEA, you have the following rights:

  • Access: View and export all your data via Settings → Export my data
  • Rectification: Edit your profile, events, and survey responses at any time in the app
  • Erasure: Permanently delete your account and all data via Settings → Delete account
  • Portability: Export your data in JSON format via Settings → Export my data
  • Restrict processing: Disable specific processing via Settings → Privacy & Data
  • Withdraw consent: Toggle any optional consent off at any time in Settings → Privacy & Data
  • Lodge a complaint: Contact the Polish DPA at uodo.gov.pl

To exercise rights not available in-app, contact us at mindcue@pm.me.

8. Consent management

All optional processing is controlled by you. During onboarding and at any time via Settings → Privacy & Data, you can independently toggle:

ConsentWhat it controlsRequired?

The app functions fully without any optional consent — calendar, manual tips, and journalling all work without AI or analytics enabled.

9. Children's privacy

Mind Cue is not intended for users under 16. We do not knowingly collect data from children. If you believe a child has provided data, contact mindcue@pm.me and we will delete it promptly.

10. Security

  • All data is encrypted in transit (TLS 1.2+) and at rest (AES-256 / AES-GCM)
  • Authentication uses Sign in with Apple — no passwords are stored by us
  • ABC journal entries are encrypted on-device using AES-GCM before storage
  • Firestore security rules enforce strict per-user data isolation
  • The Claude API key is server-side only — never shipped in the app binary

11. Changes to this policy
